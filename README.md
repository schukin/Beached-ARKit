Beached-ARKit
=============

This is a dumb ARKit demo.

![Demo gif](https://raw.githubusercontent.com/schukin/Beached-ARKit/master/demo.gif)

## Requirements

* Xcode 9 Beta 2
* iOS 11 Beta 2
* A9 or better chip for ARWorldTrackingSessionConfiguration
